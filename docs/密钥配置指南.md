# 密钥配置指南

## 📋 概述

本项目使用环境变量管理所有敏感配置（API密钥、数据库密码等），确保代码中不包含硬编码的敏感信息。

## 🚀 快速开始

### 1. 复制配置模板

```bash
cd backend
cp .env.example .env
```

### 2. 编辑配置文件

打开 `backend/.env` 文件，填写你的配置：

```bash
# 必需配置（如果使用火山引擎TTS）
VOLCENGINE_TTS_APP_ID=你的应用ID
VOLCENGINE_TTS_ACCESS_TOKEN=你的访问令牌
VOLCENGINE_TTS_SECRET_KEY=你的密钥

# 推荐配置（用于文本和图片生成）
VOLCENGINE_ARK_API_KEY=你的ARK API Key
```

### 3. 验证配置

启动应用时，配置验证会自动运行：

- **开发环境**：缺少配置会显示警告，但不会阻止启动
- **生产环境**：缺少必需配置会抛出异常，阻止启动

## 🔐 配置项说明

### 必需配置（生产环境）

| 配置项 | 说明 | 获取方式 |
|--------|------|----------|
| `VOLCENGINE_TTS_APP_ID` | 火山引擎TTS应用ID | 火山引擎控制台 |
| `VOLCENGINE_TTS_ACCESS_TOKEN` | 火山引擎TTS访问令牌 | 火山引擎控制台 |
| `VOLCENGINE_TTS_SECRET_KEY` | 火山引擎TTS密钥 | 火山引擎控制台 |
| `ALLOWED_ORIGINS` | CORS允许的来源（生产环境） | 你的前端域名，多个用逗号分隔 |

### 推荐配置

| 配置项 | 说明 | 获取方式 |
|--------|------|----------|
| `VOLCENGINE_ARK_API_KEY` | 火山引擎ARK API Key | 火山引擎控制台 |
| `DASHSCOPE_API_KEY` | 阿里云DashScope API Key | 阿里云控制台 |

### 可选配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `OPENAI_API_KEY` | OpenAI API Key | - |
| `ZHIPU_API_KEY` | 智谱AI API Key | - |
| `BAIDU_API_KEY` | 百度文心一言 API Key | - |
| `BAIDU_SECRET_KEY` | 百度文心一言 Secret Key | - |

## 🔒 安全最佳实践

### ✅ 推荐做法

1. **使用环境变量**：所有敏感信息都通过环境变量配置
2. **使用 .env 文件**：开发环境使用 `.env` 文件（已加入 `.gitignore`）
3. **生产环境使用系统环境变量**：生产环境不要使用 `.env` 文件，直接设置系统环境变量
4. **定期轮换密钥**：定期更新API密钥
5. **最小权限原则**：只授予必要的权限

### ❌ 禁止做法

1. **不要硬编码密钥**：代码中不要包含任何密钥
2. **不要提交 .env 文件**：确保 `.env` 在 `.gitignore` 中
3. **不要分享密钥**：不要在聊天、邮件、文档中分享密钥
4. **不要使用默认密钥**：不要使用示例密钥或测试密钥

## 📝 配置验证

### 手动验证

```python
from utils.config_validator import ConfigValidator

# 验证配置
is_valid, errors, warnings = ConfigValidator.validate()

# 打印报告
ConfigValidator.print_validation_report()
```

### 自动验证

在 `backend/config.py` 中取消注释以下行：

```python
from utils.config_validator import validate_config_on_startup
validate_config_on_startup()
```

## 🔧 故障排查

### 问题1：应用启动失败，提示缺少配置

**解决方案**：
1. 检查 `.env` 文件是否存在
2. 检查环境变量是否正确设置
3. 检查配置项名称是否正确（区分大小写）

### 问题2：TTS功能不可用

**可能原因**：
- `VOLCENGINE_TTS_APP_ID` 未设置
- `VOLCENGINE_TTS_ACCESS_TOKEN` 未设置
- `VOLCENGINE_TTS_SECRET_KEY` 未设置

**解决方案**：
1. 检查 `.env` 文件中是否设置了上述三个配置项
2. 检查配置值是否正确（去除首尾空格）
3. 检查火山引擎控制台中的密钥是否有效

### 问题3：图片生成功能不可用

**可能原因**：
- `VOLCENGINE_ARK_API_KEY` 未设置

**解决方案**：
1. 在 `.env` 文件中设置 `VOLCENGINE_ARK_API_KEY`
2. 检查API Key是否有效
