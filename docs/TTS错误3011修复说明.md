# TTS错误3011修复说明

**错误信息**：
```
火山引擎语音合成失败: HTTP 400, {"reqid":"tts_1770998984_7437","code":3011,"operation":"query","message":"illegal input text!","sequence":null}
```

**错误代码**: 3011 - illegal input text

---

## ✅ 已完成的修复

### 1. 添加了文本清理函数

在 `backend/models/voice_model_service.py` 中添加了 `_clean_text_for_volcengine()` 方法：

**功能**：
- ✅ 去除首尾空白字符
- ✅ 移除控制字符（保留换行符、制表符、回车符）
- ✅ 移除零宽字符（可能影响API解析）
- ✅ 限制文本长度（最大5000字符）
- ✅ 验证文本不为空

### 2. 改进了错误处理

**改进内容**：
- ✅ 特殊处理错误代码3011（文本格式错误）
- ✅ 特殊处理错误代码3001（认证失败）
- ✅ 特殊处理错误代码3002（参数错误）
- ✅ 更详细的错误信息

### 3. 添加了日志记录

**改进内容**：
- ✅ 记录文本清理前后的长度
- ✅ 记录文本验证失败时的原始文本
- ✅ 使用统一的logger系统

---

## 🔍 问题原因

错误3011通常由以下原因引起：

1. **文本为空或只包含空白字符**
2. **文本包含控制字符**（如ASCII 0-31中的某些字符）
3. **文本包含零宽字符**（如零宽空格、零宽连字符等）
4. **文本长度超出限制**
5. **文本编码问题**

---

## 📝 修复后的处理流程

1. **文本清理**：
   ```python
   cleaned_text = self._clean_text_for_volcengine(text)
   ```
   - 去除首尾空白
   - 移除控制字符
   - 移除零宽字符
   - 限制长度

2. **API调用**：
   - 使用清理后的文本
   - 发送到火山引擎API

3. **错误处理**：
   - 如果返回3011错误，会显示详细的错误信息
   - 提示检查文本格式
