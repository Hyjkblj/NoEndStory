### 5. **缺少统一的日志系统**

**问题**：

- 日志级别不统一
- 无法控制日志输出
- 生产环境难以排查问题

```python
# backend/utils/logger.py
import logging
import sys
from pythonjsonlogger import jsonlogger

def setup_logger(name: str, level: str = "INFO") -> logging.Logger:
    """设置结构化日志"""
    logger = logging.getLogger(name)
    logger.setLevel(getattr(logging, level))
    
    # 控制台输出（开发环境）
    console_handler = logging.StreamHandler(sys.stdout)
    formatter = jsonlogger.JsonFormatter(
        '%(asctime)s %(name)s %(levelname)s %(message)s'
    )
    console_handler.setFormatter(formatter)
    logger.addHandler(console_handler)
    
    return logger

# 使用示例
logger = setup_logger(__name__)
logger.info("图片生成成功", extra={"character_id": 123, "image_url": "..."})
```

### 6. **异常处理不统一**

**问题**：
- 各Service的错误处理方式不一致
- Router层重复的错误处理代码
- 缺少统一的异常类

**问题代码**：

```python

try:
    result = service.method()
except ValueError as e:
    return error_response(code=400, message=str(e))
except Exception as e:
    import traceback
    traceback.print_exc()
    return error_response(code=500, message=str(e))
```

**改进方案**：

```python
# backend/api/exceptions.py
class ServiceException(Exception):
    """服务层异常基类"""
    def __init__(self, message: str, code: int = 500, details: dict = None):
        self.message = message
        self.code = code
        self.details = details or {}
        super().__init__(self.message)

class ImageGenerationException(ServiceException):
    """图片生成异常"""
    pass

class CharacterNotFoundException(ServiceException):
    """角色不存在异常"""
    def __init__(self, character_id: int):
        super().__init__(f"角色不存在: {character_id}", code=404)

# backend/api/middleware/error_handler.py
from fastapi import Request
from fastapi.responses import JSONResponse
from api.exceptions import ServiceException

@app.exception_handler(ServiceException)
async def service_exception_handler(request: Request, exc: ServiceException):
    return JSONResponse(
        status_code=exc.code,
        content={
            "error": exc.message,
            "code": exc.code,
            "details": exc.details
        }
    )
```

### 11. **循环中同步调用API（性能问题）** 🟡 中优先级

**问题代码**：

```python
# backend/api/services/image_service.py (第548-615行)
for i in range(group_count):  # ❌ 循环中同步调用API
    response = requests.post(
        self.volcengine_api_url,
        headers=headers,

    )

```

**问题**：
- 生成3张组图需要串行调用3次API
- 总耗时 = 单次耗时 × 3（可能120秒+）
- 阻塞主线程

**改进方案**：

```python
# 使用并发请求
import asyncio
import aiohttp

async def _generate_group_images_async(self, prompts: List[str], ...):
    """并发生成多张图片"""
    async with aiohttp.ClientSession() as session:
        tasks = [
            self._generate_single_image_async(session, prompt, ...)
            for prompt in prompts
        ]
        results = await asyncio.gather(*tasks)
        return [r for r in results if r]
```

**收益**：
- 3张图片并发生成，总耗时 ≈ 单次耗时（节省60-80秒）

**优先级**：🟡 中

```python
def safe_write_file(filepath: str, content: bytes) -> bool:
    """安全写入文件"""
    try:
        os.makedirs(os.path.dirname(filepath), exist_ok=True)
        with open(filepath, 'wb') as f:
            f.write(content)
        return True
    except Exception as e:
```
