# 语音功能实现方案

## 📋 方案概述

基于文档分析和当前系统架构，推荐采用 **Hybrid混合式方案**（固定语音包 + 动态TTS），结合情绪分析和状态系统，实现沉浸式语音体验。

---

## 🎯 核心设计原则

1. **即时TTS为主**：适合AI生成的动态剧情
2. **固定语音包为辅**：常用语气词、音效增强沉浸感
3. **情绪驱动**：基于角色状态值动态调整语音参数
4. **智能缓存**：预缓存常用语音，降低延迟和成本
5. **状态协同**：语音与角色好感度、情绪值等状态联动

## 📦 技术选型

### TTS提供商（按优先级）

#### 1. **OpenAI TTS**（推荐用于开发阶段）
- ✅ 成本低（$15/1M字符）
- ✅ 质量高，自然流畅
- ✅ 支持多种语言和声音
- ✅ API简单易用
- ⚠️ 不支持情绪控制（需配合情绪分析）

#### 2. **ElevenLabs**（推荐用于生产环境）
- ✅ 情绪表达最强
- ✅ 角色一致性好
- ✅ 支持Voice Cloning
- ⚠️ 成本较高（$5/1K字符）

#### 3. **Azure TTS**（备选）
- ✅ 商业化成熟
- ✅ 支持SSML（情绪控制）
- ✅ 成本适中
- ⚠️ 中文支持一般

#### 4. **国内方案**（备选）
- MiniMax TTS：多风格，成本低
- 火山引擎TTS：表达不错
- 阿里SenseVoice：情绪较好

**建议**：开发阶段用OpenAI TTS，生产环境用ElevenLabs或Azure TTS

import os
import hashlib
import json
from typing import Optional, Dict, Any
from openai import OpenAI
from pathlib import Path
import config

class TTSService:
    """TTS语音合成服务（支持多种提供商）"""
    
    def __init__(self):
        self.provider = os.getenv('TTS_PROVIDER', 'openai')  # openai, elevenlabs, azure
        self.cache_dir = Path(config.BASE_DIR) / 'backend' / 'audio' / 'cache'
        self.cache_dir.mkdir(parents=True, exist_ok=True)
        
        # 初始化客户端
        if self.provider == 'openai':
            self.client = OpenAI(api_key=os.getenv('OPENAI_API_KEY'))
            self.model = 'tts-1'  # 或 tts-1-hd（更高质量）
            self.voice = 'alloy'  # alloy, echo, fable, onyx, nova, shimmer
        elif self.provider == 'elevenlabs':
            # TODO: 实现ElevenLabs客户端
            pass
        elif self.provider == 'azure':
            # TODO: 实现Azure TTS客户端
            pass
    
    def generate_speech(
        self,
        text: str,
        character_id: int,
        emotion_params: Optional[Dict[str, Any]] = None,
        use_cache: bool = True
    ) -> Dict[str, Any]:
        """生成语音
        
        Args:
            text: 要合成的文本
            character_id: 角色ID（用于选择声音）
            emotion_params: 情绪参数
                {
                    'emotion': 'happy',  # 情绪类型
                    'speed': 1.0,        # 语速（0.5-2.0）
                    'tone': 'soft',     # 语调
                    'volume': 0.8,      # 音量（0-1）
                    'pause_before': 0,  # 前停顿（ms）
                    'pause_after': 0    # 后停顿（ms）
                }
            use_cache: 是否使用缓存
        
        Returns:
            {
                'audio_url': '/static/audio/xxx.mp3',
                'audio_path': '/path/to/audio.mp3',
                'duration': 3.5,  # 音频时长（秒）
                'cached': True     # 是否来自缓存
            }
        """
        # 生成缓存键
        cache_key = self._generate_cache_key(text, character_id, emotion_params)
        cache_path = self.cache_dir / f"{cache_key}.mp3"
        
        # 检查缓存
        if use_cache and cache_path.exists():
            return {
                'audio_url': f'/static/audio/cache/{cache_key}.mp3',
                'audio_path': str(cache_path),
                'cached': True
            }
        
        # 生成语音
        if self.provider == 'openai':
            audio_data = self._generate_openai_speech(text, character_id, emotion_params)
        elif self.provider == 'elevenlabs':
            audio_data = self._generate_elevenlabs_speech(text, character_id, emotion_params)
        else:
            raise ValueError(f"不支持的TTS提供商: {self.provider}")
        
        # 保存到缓存
        cache_path.write_bytes(audio_data)
        
        return {
            'audio_url': f'/static/audio/cache/{cache_key}.mp3',
            'audio_path': str(cache_path),
            'duration': self._get_audio_duration(cache_path),
            'cached': False
        }
    
    def _generate_openai_speech(
        self,
        text: str,
        character_id: int,
        emotion_params: Optional[Dict[str, Any]] = None
    ) -> bytes:
        """使用OpenAI TTS生成语音"""
        # 根据角色ID选择声音（可以从数据库读取角色声音配置）
        voice = self._get_character_voice(character_id)
        
        # 调用OpenAI TTS API
        response = self.client.audio.speech.create(
            model=self.model,
            voice=voice,
            input=text,
            speed=self._calculate_speed(emotion_params) if emotion_params else 1.0
        )
        
        return response.content
    
    def _get_character_voice(self, character_id: int) -> str:
        """根据角色ID获取声音配置"""
        # TODO: 从数据库读取角色的声音配置
        # 可以根据角色性别、性格等选择不同的声音
        # 默认返回alloy
        return 'alloy'
    
    def _calculate_speed(self, emotion_params: Dict[str, Any]) -> float:
        """根据情绪参数计算语速"""
        base_speed = emotion_params.get('speed', 1.0)
        # 情绪影响语速
        emotion = emotion_params.get('emotion', 'neutral')
        speed_modifiers = {
            'excited': 1.2,
            'anxious': 1.1,
            'sad': 0.8,
            'calm': 0.9,
            'neutral': 1.0
        }
        modifier = speed_modifiers.get(emotion, 1.0)
        return base_speed * modifier
    
    def _generate_cache_key(
        self,
        text: str,
        character_id: int,
        emotion_params: Optional[Dict[str, Any]] = None
    ) -> str:
        """生成缓存键"""
        key_data = {
            'text': text,
            'character_id': character_id,
            'emotion_params': emotion_params or {}
        }
        key_str = json.dumps(key_data, sort_keys=True, ensure_ascii=False)
        return hashlib.md5(key_str.encode('utf-8')).hexdigest()

#### 1.2 创建情绪分析服务 (`backend/api/services/emotion_service.py`)

```python
"""情绪分析服务"""
from typing import Dict, Any, Optional
from llm import LLMService

class EmotionService:
    """情绪分析服务（分析对话文本的情绪，结合角色状态值）"""
    
    def __init__(self):
        self.llm_service = LLMService()
    
    def analyze_emotion(
        self,
        text: str,
        character_state: Dict[str, float],
        character_personality: Dict[str, Any]
    ) -> Dict[str, Any]:
        """分析对话文本的情绪
        
        Args:
            text: 对话文本
            character_state: 角色状态值
                {
                    'favorability': 50,
                    'trust': 60,
                    'happiness': 70,
                    'stress': 20,
                    ...
                }
            character_personality: 角色性格信息
        
        Returns:
            {
                'emotion': 'happy',      # 主要情绪
                'emotions': ['happy', 'warm'],  # 复合情绪
                'speed': 1.0,            # 语速（0.5-2.0）
                'tone': 'soft',          # 语调（soft, normal, strong）
                'volume': 0.8,           # 音量（0-1）
                'pause_before': 300,     # 前停顿（ms）
                'pause_after': 200       # 后停顿（ms）
            }
        """
        # 构建情绪分析提示词
        prompt = self._build_emotion_prompt(text, character_state, character_personality)
        
        # 调用LLM分析情绪
        try:
            response = self.llm_service.chat_completion(
                prompt=prompt,
                max_tokens=200,
                temperature=0.3  # 低温度保证一致性
            )
            
            # 解析LLM返回的情绪信息
            emotion_data = self._parse_emotion_response(response)
        except Exception as e:
            print(f"[情绪分析错误] {e}")
            # 回退到基于状态值的简单情绪分析
            emotion_data = self._fallback_emotion_analysis(character_state)
        
        return emotion_data
    
    def _build_emotion_prompt(
        self,
        text: str,
        character_state: Dict[str, float],
        character_personality: Dict[str, Any]
    ) -> str:
        """构建情绪分析提示词"""
        favorability = character_state.get('favorability', 0)
        happiness = character_state.get('happiness', 50)
        stress = character_state.get('stress', 0)
        confidence = character_state.get('confidence', 50)
        
        return f"""分析以下角色对话的情绪和语音参数。

【角色性格】：
{character_personality.get('personality_text', '')}

【角色当前状态】：
- 好感度：{favorability}
- 快乐值：{happiness}
- 压力值：{stress}
- 自信度：{confidence}

【对话文本】：
"{text}"

请分析这段对话的情绪，并返回JSON格式：
{{
    "emotion": "主要情绪（happy/sad/angry/shy/anxious/excited/calm/neutral等）",
    "emotions": ["情绪1", "情绪2"],  // 复合情绪列表
    "speed": 1.0,  // 语速（0.5-2.0，1.0为正常）
    "tone": "soft",  // 语调（soft/normal/strong）
    "volume": 0.8,  // 音量（0-1）
    "pause_before": 300,  // 说话前停顿（毫秒）
    "pause_after": 200  // 说话后停顿（毫秒）
}}

只返回JSON，不要其他内容。"""
    
    def _parse_emotion_response(self, response: str) -> Dict[str, Any]:
        """解析LLM返回的情绪信息"""
        import json
        import re
        
        # 尝试提取JSON
        json_match = re.search(r'\{[^}]+\}', response, re.DOTALL)
        if json_match:
            try:
                return json.loads(json_match.group())
            except:
                pass
        
        # 如果解析失败，返回默认值
        return {
            'emotion': 'neutral',
            'emotions': ['neutral'],
            'speed': 1.0,
            'tone': 'normal',
            'volume': 0.8,
            'pause_before': 0,
            'pause_after': 0
        }
    
    def _fallback_emotion_analysis(self, character_state: Dict[str, float]) -> Dict[str, Any]:
        """基于状态值的简单情绪分析（回退方案）"""
        favorability = character_state.get('favorability', 0)
        happiness = character_state.get('happiness', 50)
        stress = character_state.get('stress', 0)
        
        # 根据状态值推断情绪
        if happiness > 70:
            emotion = 'happy'
            speed = 1.1
        elif happiness < 30:
            emotion = 'sad'
            speed = 0.8
        elif stress > 70:
            emotion = 'anxious'
            speed = 1.2
        elif favorability > 60:
            emotion = 'warm'
            speed = 1.0
        elif favorability < -30:
            emotion = 'cold'
            speed = 0.9
        else:
            emotion = 'neutral'
            speed = 1.0
        
        return {
            'emotion': emotion,
            'emotions': [emotion],
            'speed': speed,
            'tone': 'normal',
            'volume': 0.8,
            'pause_before': 0,
            'pause_after': 0
        }
```

```typescript
import React, { useEffect, useRef, useState } from 'react';
import { Button } from 'antd';
import { SoundOutlined, PauseOutlined } from '@ant-design/icons';
import './AudioPlayer.css';

interface AudioPlayerProps {
  audioUrl?: string;
  autoPlay?: boolean;
  onEnded?: () => void;
  pauseBefore?: number;  // 播放前停顿（ms）
  pauseAfter?: number;   // 播放后停顿（ms）
  volume?: number;        // 音量（0-1）
}

const AudioPlayer: React.FC<AudioPlayerProps> = ({
  audioUrl,
  autoPlay = true,
  onEnded,
  pauseBefore = 0,
  pauseAfter = 0,
  volume = 1.0
}) => {
  const audioRef = useRef<HTMLAudioElement>(null);
  const [isPlaying, setIsPlaying] = useState(false);
  const [isLoading, setIsLoading] = useState(false);

  useEffect(() => {
    if (!audioUrl || !audioRef.current) return;

    const audio = audioRef.current;
    audio.volume = volume;

    const handlePlay = () => setIsPlaying(true);
    const handlePause = () => setIsPlaying(false);
    const handleEnded = () => {
      setIsPlaying(false);
      if (onEnded) {
        // 播放后停顿
        setTimeout(() => {
          onEnded();
        }, pauseAfter);
      }
    };

    audio.addEventListener('play', handlePlay);
    audio.addEventListener('pause', handlePause);
    audio.addEventListener('ended', handleEnded);

    return () => {
      audio.removeEventListener('play', handlePlay);
      audio.removeEventListener('pause', handlePause);
      audio.removeEventListener('ended', handleEnded);
    };
  }, [audioUrl, volume, pauseAfter, onEnded]);

  useEffect(() => {
    if (!audioUrl || !autoPlay) return;

    const playAudio = async () => {
      if (!audioRef.current) return;

      setIsLoading(true);
      
      // 播放前停顿
      await new Promise(resolve => setTimeout(resolve, pauseBefore));

      try {
        await audioRef.current.play();
        setIsLoading(false);
      } catch (error) {
        console.error('播放音频失败:', error);
        setIsLoading(false);
      }
    };

    playAudio();
  }, [audioUrl, autoPlay, pauseBefore]);

  const handleTogglePlay = () => {
    if (!audioRef.current) return;

    if (isPlaying) {
      audioRef.current.pause();
    } else {
      audioRef.current.play();
    }
  };

  if (!audioUrl) return null;

  return (
    <div className="audio-player">
      <audio ref={audioRef} src={audioUrl} preload="auto" />
      <Button
        type="text"
        icon={isPlaying ? <PauseOutlined /> : <SoundOutlined />}
        onClick={handleTogglePlay}
        loading={isLoading}
        className="audio-control-button"
      />
    </div>
  );
};

export default AudioPlayer;
```

# backend/api/services/sound_effect_service.py

class SoundEffectService:
    """固定音效服务"""
    
    SOUND_EFFECTS = {
        'emotion': {
            'sigh': '/static/audio/effects/sigh.mp3',
            'laugh': '/static/audio/effects/laugh.mp3',
            'cry': '/static/audio/effects/cry.mp3',
            'gasp': '/static/audio/effects/gasp.mp3',
        },
        'scene': {
            'footstep': '/static/audio/effects/footstep.mp3',
            'door_open': '/static/audio/effects/door_open.mp3',
        }
    }
    
    def get_sound_effect(self, emotion: str) -> Optional[str]:
        """根据情绪获取音效"""
        return self.SOUND_EFFECTS['emotion'].get(emotion)

- 位置：`backend/audio/cache/`
- 命名：MD5(text + character_id + emotion_params)
- 清理：定期清理未使用的缓存（LRU策略）

### 2. Redis缓存（可选）
- 存储：音频URL和元数据
- TTL：7天
- 用途：跨服务器共享缓存

### 3. CDN缓存（生产环境）
- 存储：常用语音文件
- 预热：预生成常用对话的语音
