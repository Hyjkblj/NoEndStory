# 玩家自定义音色实现方案

## 📋 方案概述

支持玩家为DIY角色自定义音色，提供**三种安全可控的方式**：
1. **预设音色库选择**（最安全，推荐）
2. **TTS参数调整**（中等自由度）
3. **音频样本克隆**（最高自由度，需审核）

## 🇨🇳 国产TTS模型推荐

基于预设音色库方案，推荐使用以下国产TTS模型（详见 `docs/国产TTS模型推荐.md`）：

### 🥇 首选：火山引擎 - 豆包语音合成大模型
- ✅ **情绪表达最强**：基于大模型能力，智能预测文本情绪、语调
- ✅ **声音复刻**：支持秒级快速定制专属音色
- ✅ **超自然语音**：韵律自然，拟人度高
- ✅ **成本适中**：￥22.50/10万字包（新客优惠）
- 📖 文档：https://www.volcengine.com/docs/6561/1257584

### 🥈 备选：阿里云 - 通义千问语音合成
- ✅ **项目已集成**：可复用现有通义千问配置
- ✅ **统一平台**：与文本生成使用同一平台
- ✅ **SSML支持**：支持精细控制语音参数
- 📖 文档：https://help.aliyun.com/zh/dashscope/

### 🥉 其他选项：腾讯云、百度
- 腾讯云：音色丰富，商业化成熟
- 百度：成本较低，项目已配置

**详细对比和集成方案请参考**：`docs/国产TTS模型推荐.md`

## 🎯 三种实现方式

### 方式一：预设音色库选择（推荐，最安全）

**实现方式**：
- 提供20-30种预设音色（男声/女声/中性声）
- 玩家在创建角色时选择
- 支持预览试听
- 零风险，零审核成本

**优势**：
- ✅ 完全安全，无版权风险
- ✅ 无需审核
- ✅ 成本低
- ✅ 体验稳定

**适用场景**：默认方案，所有玩家可用

---

### 方式二：TTS参数调整（中等自由度）

**实现方式**：
- 基于预设音色，允许调整参数：
  - 音调（pitch）：-5到+5
  - 语速（speed）：0.5到2.0
  - 音色亮度（brightness）：0到1
  - 音色厚度（thickness）：0到1
- 实时预览效果
- 保存为角色专属配置

**优势**：
- ✅ 安全可控
- ✅ 个性化程度高
- ✅ 无需审核
- ✅ 成本低

**适用场景**：进阶玩家，想要更多个性化

---

### 方式三：音频样本克隆（最高自由度，需审核）

**实现方式**：
- 玩家上传15-30秒音频样本
- 系统使用Voice Cloning技术克隆音色
- **必须审核**：检测是否为名人/声优声音
- 通过审核后保存为角色专属音色

**优势**：
- ✅ 最高自由度
- ✅ 完全个性化
- ⚠️ 需要审核机制
- ⚠️ 成本较高

**适用场景**：高级功能，需要付费或VIP权限

```python
# backend/models/character.py

class CharacterVoice(Base):
    """角色音色配置表"""
    __tablename__ = 'character_voices'
    
    id = Column(Integer, primary_key=True, autoincrement=True)
    character_id = Column(Integer, ForeignKey('characters.id'), nullable=False, unique=True)
    
    # 音色类型：preset（预设）、custom（自定义参数）、cloned（克隆）
    voice_type = Column(String(20), nullable=False, default='preset')
    
    # 预设音色ID（如果使用预设音色库）
    preset_voice_id = Column(String(50), nullable=True)
    
    # TTS参数调整（JSON格式）
    voice_params = Column(JSON, nullable=True, default={
        'pitch': 0,        # 音调偏移（-5到+5）
        'speed': 1.0,      # 语速（0.5-2.0）
        'brightness': 0.5, # 音色亮度（0-1）
        'thickness': 0.5,  # 音色厚度（0-1）
    })
    
    # 克隆音色配置（如果使用Voice Cloning）
    cloned_voice_id = Column(String(100), nullable=True)  # ElevenLabs voice_id
    cloned_voice_name = Column(String(100), nullable=True)
    audio_sample_path = Column(String(500), nullable=True)  # 原始音频样本路径
    
    # 审核状态（仅克隆音色需要）
    review_status = Column(String(20), nullable=True, default='pending')  # pending, approved, rejected
    review_reason = Column(Text, nullable=True)  # 审核原因
    
    created_at = Column(DateTime, default=datetime.utcnow)
    updated_at = Column(DateTime, default=datetime.utcnow, onupdate=datetime.utcnow)
    
    # 关联关系
    character = relationship('Character', back_populates='voice')
```

```python
"""音色审核服务（检测名人声音、版权内容）"""

import os
from typing import Dict, Any, Optional
from llm import LLMService

class VoiceReviewService:
    """音色审核服务"""
    
    def __init__(self):
        self.llm_service = LLMService()
        # 名人声音黑名单（可以扩展）
        self.celebrity_blacklist = [
            '周杰伦', '邓紫棋', '林俊杰', '王力宏',
            'Taylor Swift', 'Adele', 'Ed Sheeran',
            # ... 更多名人
        ]
    
    def review_audio_sample(
        self,
        audio_path: str,
        user_id: Optional[int] = None,
        character_id: Optional[int] = None
    ) -> Dict[str, Any]:
        """审核音频样本
        
        Returns:
            {
                'status': 'pending' | 'approved' | 'rejected',
                'reason': '审核原因',
                'risk_level': 'low' | 'medium' | 'high',
                'requires_manual_review': True/False
            }
        """
        # 1. 基础检查（文件大小、格式等）
        basic_check = self._basic_check(audio_path)
        if not basic_check['valid']:
            return {
                'status': 'rejected',
                'reason': basic_check['reason'],
                'risk_level': 'high',
                'requires_manual_review': False
            }
        
        # 2. 使用AI检测是否为名人声音
        celebrity_check = self._check_celebrity_voice(audio_path)
        if celebrity_check['is_celebrity']:
            return {
                'status': 'rejected',
                'reason': f'检测到疑似名人声音：{celebrity_check["detected_name"]}',
                'risk_level': 'high',
                'requires_manual_review': False
            }
        
        # 3. 检测音频质量
        quality_check = self._check_audio_quality(audio_path)
        if not quality_check['suitable']:
            return {
                'status': 'rejected',
                'reason': quality_check['reason'],
                'risk_level': 'medium',
                'requires_manual_review': False
            }
        
        # 4. 低风险样本自动通过，高风险样本需要人工审核
        risk_level = self._calculate_risk_level(audio_path, celebrity_check, quality_check)
        
        if risk_level == 'low':
            return {
                'status': 'approved',
                'reason': '自动审核通过',
                'risk_level': 'low',
                'requires_manual_review': False
            }
        else:
            return {
                'status': 'pending',
                'reason': '需要人工审核',
                'risk_level': risk_level,
                'requires_manual_review': True
            }

        """使用AI检测是否为名人声音"""

        # 方案2：使用声音特征匹配（需要建立名人声音库）
        # 方案3：使用LLM分析音频描述
        
        # 这里使用LLM分析（简化版）
        try:
            # 先转文字（使用语音识别）
            # text = self._speech_to_text(audio_path)
            
            # 使用LLM分析
            prompt = f"""分析以下音频样本，判断是否可能是名人或声优的声音。

请返回JSON格式：
{{
    "is_celebrity": true/false,
    "detected_name": "检测到的名字（如果有）",
    "confidence": 0.0-1.0,
    "reason": "判断原因"
}}

只返回JSON，不要其他内容。"""
            
            response = self.llm_service.chat_completion(prompt, max_tokens=200)
            # 解析响应...
            
            # 简化版：返回默认值
            return {
                'is_celebrity': False,
                'detected_name': None,
                'confidence': 0.0
            }
        except Exception as e:
            print(f"[审核错误] {e}")
            return {
                'is_celebrity': False,
                'detected_name': None,
                'confidence': 0.0
            }
    
    def _basic_check(self, audio_path: str) -> Dict[str, Any]:
        """基础检查（文件大小、格式、时长）"""
        import os
        from pathlib import Path
        
        path = Path(audio_path)
        if not path.exists():
```

```python
# backend/api/services/voice_cloning_service.py

"""Voice Cloning服务（使用ElevenLabs）"""

import os
from typing import Dict, Any, Optional
from elevenlabs import ElevenLabs, VoiceSettings

class VoiceCloningService:
    """Voice Cloning服务"""
    
    def __init__(self):
        self.api_key = os.getenv('ELEVENLABS_API_KEY')
        if not self.api_key:
            raise ValueError("ELEVENLABS_API_KEY未配置")
        
        self.client = ElevenLabs(api_key=self.api_key)
    
    def clone_voice(
        self,
        audio_path: str,
        voice_name: str,
        description: Optional[str] = None
    ) -> Dict[str, Any]:
        """克隆声音
        
        Args:
            audio_path: 音频样本路径
            voice_name: 音色名称
            description: 音色描述
        
        Returns:
            {
                'voice_id': 'ElevenLabs voice ID',
                'voice_name': voice_name,
                'success': True
            }
        """
        try:
            # 读取音频文件
            with open(audio_path, 'rb') as audio_file:
                # 调用ElevenLabs API克隆声音
                voice = self.client.clone(
                    name=voice_name,
                    description=description or f"Cloned voice: {voice_name}",
                    files=[audio_file]
                )
                
                return {
                    'voice_id': voice.voice_id,
                    'voice_name': voice_name,
                    'success': True
                }
        except Exception as e:
            print(f"[Voice Cloning错误] {e}")
            return {
                'success': False,
                'error': str(e)
            }
    
    def delete_voice(self, voice_id: str) -> bool:
        """删除克隆的声音"""
        try:
            self.client.voices.delete(voice_id)
            return True
        except Exception as e:
            print(f"[删除声音错误] {e}")
            return False
```

```python
@router.post("/clone/upload")
async def upload_voice_sample(
    character_id: int,
    audio_file: UploadFile = File(...),
    voice_name: Optional[str] = None
):
    """上传音频样本进行Voice Cloning（需要审核）"""
    import tempfile
    import os
    
    # 保存上传的文件
    with tempfile.NamedTemporaryFile(delete=False, suffix='.mp3') as tmp_file:
        content = await audio_file.read()
        tmp_file.write(content)
        tmp_path = tmp_file.name
    
    try:
        # 审核音频样本
        review_result = review_service.review_audio_sample(
            audio_path=tmp_path,
            character_id=character_id
        )
        
        if review_result['status'] == 'rejected':
            os.unlink(tmp_path)
            raise HTTPException(
                status_code=400,
                detail=f"审核未通过：{review_result['reason']}"
            )
        
        # 保存音频样本
        sample_dir = os.path.join('backend', 'audio', 'samples', str(character_id))
        os.makedirs(sample_dir, exist_ok=True)
        sample_path = os.path.join(sample_dir, f"{character_id}_sample.mp3")
        os.rename(tmp_path, sample_path)
        
        # 如果需要人工审核，先保存待审核状态
        if review_result['requires_manual_review']:
            db_manager.set_character_voice(
                character_id=character_id,
                voice_type='cloned',
                review_status='pending',
                audio_sample_path=sample_path
            )
            return {
                "success": True,
                "status": "pending",
                "message": "音频已上传，等待人工审核"
            }
        
        # 自动审核通过，进行Voice Cloning
        clone_result = cloning_service.clone_voice(
            audio_path=sample_path,
            voice_name=voice_name or f"Character_{character_id}"
        )
        
        if not clone_result['success']:
            raise HTTPException(status_code=500, detail="Voice Cloning失败")
        
        # 保存克隆结果
        db_manager.set_character_voice(
            character_id=character_id,
            voice_type='cloned',
            cloned_voice_id=clone_result['voice_id'],
            cloned_voice_name=clone_result['voice_name'],
            audio_sample_path=sample_path,
            review_status='approved'
        )
        
        return {
            "success": True,
            "status": "approved",
            "voice_id": clone_result['voice_id'],
            "message": "音色克隆成功"
        }
    
    except HTTPException:
        raise
    except Exception as e:
        if os.path.exists(tmp_path):
            os.unlink(tmp_path)
        raise HTTPException(status_code=500, detail=str(e))
```

```typescript
import React, { useState, useEffect } from 'react';
import { Radio, Slider, Button, Upload, message, Card } from 'antd';
import { SoundOutlined, UploadOutlined } from '@ant-design/icons';
import type { UploadProps } from 'antd';
import './VoiceSelector.css';

interface VoiceSelectorProps {
  characterId?: number;
  gender: 'male' | 'female' | 'neutral';
  onVoiceSelected?: (voiceConfig: VoiceConfig) => void;
}

interface VoiceConfig {
  voiceType: 'preset' | 'custom' | 'cloned';
  presetVoiceId?: string;
  voiceParams?: {
    pitch: number;
    speed: number;
    brightness: number;
    thickness: number;
  };
}

const VoiceSelector: React.FC<VoiceSelectorProps> = ({
  characterId,
  gender,
  onVoiceSelected
}) => {
  const [voiceType, setVoiceType] = useState<'preset' | 'custom' | 'cloned'>('preset');
  const [presetVoices, setPresetVoices] = useState<any[]>([]);
  const [selectedPresetId, setSelectedPresetId] = useState<string>('');
  const [voiceParams, setVoiceParams] = useState({
    pitch: 0,
    speed: 1.0,
    brightness: 0.5,
    thickness: 0.5,
  });
  const [previewAudio, setPreviewAudio] = useState<HTMLAudioElement | null>(null);

  // 加载预设音色列表
  useEffect(() => {
    loadPresetVoices();
  }, [gender]);

  const loadPresetVoices = async () => {
    try {
      const response = await fetch(`/api/voice/presets?gender=${gender}`);
      const data = await response.json();
      setPresetVoices(data.voices[gender] || []);
      if (data.voices[gender]?.length > 0) {
        setSelectedPresetId(data.voices[gender][0].id);
      }
    } catch (error) {
      console.error('加载预设音色失败:', error);
    }
  };

  const playPreview = async (voiceId: string) => {
    try {
      const response = await fetch(`/api/voice/presets/${voiceId}/preview`);
      const data = await response.json();
      
      if (previewAudio) {
        previewAudio.pause();
      }
      
      const audio = new Audio(data.preview_audio);
      audio.play();
      setPreviewAudio(audio);
    } catch (error) {
      message.error('播放预览失败');
    }
  };

  const handleVoiceTypeChange = (e: any) => {
    setVoiceType(e.target.value);
  };

  const handlePresetSelect = (voiceId: string) => {
    setSelectedPresetId(voiceId);
    playPreview(voiceId);
  };

  const handleParamsChange = (key: string, value: number) => {
    setVoiceParams(prev => ({
      ...prev,
      [key]: value
    }));
  };

  const handleSave = () => {
    const config: VoiceConfig = {
      voiceType,
      presetVoiceId: voiceType === 'preset' ? selectedPresetId : undefined,
      voiceParams: voiceType === 'custom' ? voiceParams : undefined,
    };
    
    if (onVoiceSelected) {
      onVoiceSelected(config);
    }
    
    // 保存到后端
    if (characterId) {
      saveVoiceConfig(characterId, config);
    }
  };

  const saveVoiceConfig = async (charId: number, config: VoiceConfig) => {
    try {
      const response = await fetch('/api/voice/config', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          character_id: charId,
          voice_type: config.voiceType,
          preset_voice_id: config.presetVoiceId,
          voice_params: config.voiceParams,
        }),
      });
      
      if (response.ok) {
        message.success('音色配置已保存');
      }
    } catch (error) {
      message.error('保存失败');
    }
  };

  const uploadProps: UploadProps = {
    name: 'audio',
    action: `/api/voice/clone/upload?character_id=${characterId}`,
    accept: '.mp3,.wav,.m4a',
    beforeUpload: (file) => {
      const isAudio = ['audio/mpeg', 'audio/wav', 'audio/mp4'].includes(file.type);
      if (!isAudio) {
        message.error('只能上传音频文件！');
        return false;
      }
      const isLt10M = file.size / 1024 / 1024 < 10;
      if (!isLt10M) {
        message.error('文件大小不能超过10MB！');
        return false;
      }
      return true;
    },
    onChange: (info) => {
      if (info.file.status === 'done') {
        message.success('音频上传成功，等待审核');
      } else if (info.file.status === 'error') {
        message.error('上传失败');
      }
    },
  };

  return (
    <div className="voice-selector">
      <Radio.Group value={voiceType} onChange={handleVoiceTypeChange}>
        <Radio value="preset">预设音色</Radio>
        <Radio value="custom">自定义参数</Radio>
        <Radio value="cloned">上传音频克隆</Radio>
      </Radio.Group>

      {voiceType === 'preset' && (
        <div className="preset-voices">
          <h3>选择预设音色</h3>
          <div className="voice-grid">
            {presetVoices.map(voice => (
              <Card
                key={voice.id}
                className={`voice-card ${selectedPresetId === voice.id ? 'selected' : ''}`}
                onClick={() => handlePresetSelect(voice.id)}
              >
                <div className="voice-name">{voice.name}</div>
                <div className="voice-description">{voice.description}</div>
                <Button
                  icon={<SoundOutlined />}
                  onClick={(e) => {
                    e.stopPropagation();
                    playPreview(voice.id);
                  }}
                >
                  试听
                </Button>
              </Card>
            ))}
          </div>
        </div>
      )}

      {voiceType === 'custom' && (
        <div className="custom-params">
          <h3>调整音色参数</h3>
          <div className="param-item">
            <label>音调: {voiceParams.pitch}</label>
            <Slider
              min={-5}
              max={5}
              value={voiceParams.pitch}
              onChange={(value) => handleParamsChange('pitch', value)}
            />
          </div>
          <div className="param-item">
            <label>语速: {voiceParams.speed.toFixed(1)}</label>
            <Slider
              min={0.5}
              max={2.0}
              step={0.1}
              value={voiceParams.speed}
              onChange={(value) => handleParamsChange('speed', value)}
            />
          </div>
          <div className="param-item">
            <label>音色亮度: {voiceParams.brightness.toFixed(1)}</label>
            <Slider
              min={0}
              max={1}
              step={0.1}
              value={voiceParams.brightness}
              onChange={(value) => handleParamsChange('brightness', value)}
            />
          </div>
          <div className="param-item">
            <label>音色厚度: {voiceParams.thickness.toFixed(1)}</label>
            <Slider
              min={0}
              max={1}
              step={0.1}
              value={voiceParams.thickness}
              onChange={(value) => handleParamsChange('thickness', value)}
            />
          </div>
        </div>
      )}

      {voiceType === 'cloned' && (
        <div className="voice-cloning">
          <h3>上传音频样本（15-30秒）</h3>
          <Upload {...uploadProps}>
            <Button icon={<UploadOutlined />}>选择音频文件</Button>
          </Upload>
          <p className="hint">
            请上传15-30秒的清晰音频，系统将自动审核并克隆音色。
            注意：不能使用名人或声优的声音。
          </p>
        </div>
      )}

      <Button type="primary" onClick={handleSave} style={{ marginTop: 16 }}>
        保存音色配置
      </Button>
    </div>
  );
};

export default VoiceSelector;
```
