# 后端预设音色列表说明

本文档说明后端可用的预设音色列表，这些音色定义在 `backend/data/preset_voices.py` 中。

---

## 一、音色获取接口

**接口路径**：`GET /api/v1/tts/presets`

**说明**：
- 此接口**不依赖 TTS 服务**，直接从数据文件读取预设音色列表
- 即使 TTS 服务未启用或不可用，也能正常返回音色列表
- 仅**试听功能**（`/api/v1/tts/preview`）需要 TTS 服务启用

**参数**：
- `gender`（可选）：`male` | `female` | `neutral`，不传则返回所有

**返回格式**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "voices": [
      {
        "id": "female_001",
        "name": "通用女声",
        "description": "标准女声，适合通用场景",
        "voice_id": "BV001_streaming",
        "gender": "female",
        "style": "标准",
        "preview_text": "你好呀，很高兴见到你！"
      },
      ...
    ]
  }
}
```

## 三、音色特性

### 模型版本

- **seed-tts-2.0**：豆包语音合成模型 2.0（BV001-BV012）
- **seed-tts-1.0**：豆包语音合成模型 1.0（zh_* 系列）

### 支持的功能

- **情感支持**（`supports_emotion`）：大部分音色支持情感参数
- **混音支持**（`supports_mix`）：大部分音色支持多音色混音

### 支持的情感类型

- `happy`（开心）、`sad`（悲伤）、`angry`（愤怒）、`surprised`（惊讶）
- `neutral`（中性）、`confident`（自信）、`gentle`（温柔）、`serious`（严肃）
- `cheerful`（愉快）、`passionate`（热情）、`calm`（平静）、`dramatic`（戏剧化）、`warm`（温暖）

## 四、使用说明

### 前端获取音色列表

```typescript
import { getPresetVoices } from '@/services/api';

// 获取所有音色
const allVoices = await getPresetVoices();

// 按性别筛选（可选）
const femaleVoices = await getPresetVoices('female');
const maleVoices = await getPresetVoices('male');
```

### 设置角色音色

```typescript
import { setVoiceConfig } from '@/services/api';

await setVoiceConfig({
  character_id: 123,
  voice_type: 'preset',
  preset_voice_id: 'female_001',  // 使用预设音色 ID
});
```

### 试听音色（需要 TTS 服务启用）

```typescript
import { getVoicePreviewAudio } from '@/services/api';

const result = await getVoicePreviewAudio('female_001', '你好，这是试听。');
if (result?.audio_url) {
  // 播放音频
}
```

---

## 五、注意事项

1. **音色列表接口不依赖 TTS 服务**：即使 TTS 服务未启用，也能正常获取音色列表。
2. **试听功能需要 TTS 服务**：如果 TTS 服务未启用或不可用，试听功能会返回 503，但不影响选择音色。
3. **游戏中使用音色**：选择音色后，游戏中的对话会使用该音色进行 TTS 合成，此时需要 TTS 服务已启用。
4. **音色 ID**：前端应使用 `id` 字段（如 `female_001`），后端会将其映射到对应的 `voice_id`（如 `BV001_streaming`）。
